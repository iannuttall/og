# Agent Notes

This repo is a Cloudflare Worker template for rendering Open Graph images.

## Commands

```sh
pnpm install
pnpm dev
pnpm dev:remote
pnpm check
pnpm security:check
pnpm typecheck
pnpm test
pnpm og doctor
pnpm og render https://example.com --out og.png
```

## Stack

- Cloudflare Workers
- Hono
- Browser Rendering through `@cloudflare/puppeteer`
- R2 for persistent image cache
- Workers Cache for edge cache
- Durable Object `RenderGate` for render coordination and budgets
- TypeScript
- Biome
- Vitest

## Rules

- Use `pnpm`, not npm.
- Do not commit `.dev.vars`, `.dev.vars.*`, `.wrangler/`, `dist/`, or `node_modules/`.
- Do not add account IDs, personal routes, or secrets to committed config.
- Keep `wrangler.jsonc` public-template safe.
- Personal deploy values belong in Cloudflare build env vars such as
  `OG_ALLOWED_HOSTS`, not in committed config.
- Local personal deploy values can live in ignored `.env.deploy.local`.
- `pnpm deploy` writes ignored `wrangler.deploy.jsonc` from public deploy vars and
  deploys that ignored file. It must fail if required `OG_*` deploy values are
  missing, so template defaults cannot accidentally deploy to the wrong account.
- Generate Worker types with `pnpm types` after binding changes.
- `src/worker-configuration.d.ts` is generated by Wrangler and committed. Do
  not hand-edit it.
- Keep Browser Rendering behind cache checks and the render gate.
- Do not let public query params force a fresh render.
- `?v=` must stay ignored for public cache identity.
- Preview and purge must stay owner-only through `PURGE_TOKEN`.
- Keep `ALLOWED_HOSTS="*"` development-only.
- Avoid new storage products unless the Worker genuinely needs them.
- Update `plans/todo.md` when changing roadmap or unfinished work.

## Architecture

- `src/worker/index.ts` owns HTTP routes.
- `src/lib/og/config.ts` owns env parsing and safe defaults.
- `src/lib/og/target.ts` owns URL validation and host allowlisting.
- `src/lib/og/cache.ts` owns Workers Cache and R2.
- `src/lib/og/gate.ts` owns the Durable Object render gate.
- `src/lib/og/preflight.ts` owns cheap page validation before rendering.
- `src/lib/og/screenshot.ts` owns Browser Rendering.
- `scripts/og.mjs` owns local human-friendly commands.

## Public repo checklist

- Run `pnpm check`.
- Run `pnpm security:check`.
- Run `pnpm og doctor`.
- Run `pnpm deploy:dry-run`.
- Confirm `wrangler.jsonc` has no account-specific IDs.
- Confirm `.dev.vars` is ignored.
- Confirm GitHub repo visibility only changes after checks pass.
- Enable GitHub private vulnerability reporting after the repo is public.
